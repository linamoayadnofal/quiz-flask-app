# Quiz Flask App

This is a Flask project for creating a multiple-choice quiz (MCQ) web app.

## How to Run
1. Install dependencies:

pip install -r requirements.txt

2. Run the app:
python app.py

## Features
- Create quizzes with multiple-choice questions
- Submit answers and see the results
- Simple and user-friendly interface
